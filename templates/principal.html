Páginas disponibles:

Calcular potencia, para ello debe introducir en la url /potencia/base/exponente como se indica el siguiente ejemplo: /potencia/3/4

Contar letras de una palabra, para ello debe introducir en la url /cuenta/palabra/letra como se indica el siguiente ejemplo: /cuenta/hola/a

Biblioteca, para buscar un libro debe poner en la url /libro/codigo donde "codigo" es un número, como en el siguiente ejemplo: /libro/124
